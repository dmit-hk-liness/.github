这个场景推荐：**HKG.AS3.Pro**

**关于硬件升级计划：** DMIT 目前香港节点采用 AMD EPYC 7003 系列（AS3 平台），官方计划在 2026 年第二季度对香港和东京节点启动硬件升级，有可能换到 AN4（EPYC 9004 系列）。更重要的是，现有用户的续费价格和折扣完全不受影响，升级后也不用担心涨价。

**关于退款政策：** 购买后 3 天内（且流量使用不超过 30GB）可以申请无条件全额退款；30 天内也可以按剩余比例退款，试错成本比较低。

**支付方式：** 支付宝、微信、PayPal 全支持，有中文客服，国内用户购买门槛基本没有。
